# Codify-Web

Codify-Web is an interactive platform designed to help students master coding through structured courses, live sessions, and a gamified experience. It features a dual-leaderboard system that tracks both daily performance and all-time cumulative progress to motivate learners.

## Features
* **Gamified Leaderboards**: Tracks XP via both a Daily Leaderboard (snapshot of 24-hour activity) and an All-Time Leaderboard (long-term cumulative standing).
* **Course Management**: Structured paths for students to learn and track their progress.
* **Live Sessions**: Integration for real-time engagement between teachers and students.
* **Responsive Dashboard**: Personalized views for Admin, Teachers, and Students.

## Technologies Used
* **Frontend**: HTML5, CSS3, JavaScript.
* **Backend**: PHP 8.x.
* **Database**: MySQL.
* **Libraries**: Chart.js (for analytics and progress tracking).
